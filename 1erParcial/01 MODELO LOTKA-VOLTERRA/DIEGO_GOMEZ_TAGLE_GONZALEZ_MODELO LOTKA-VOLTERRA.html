Qu es el Modelo?

El modelo presa -depredador, tambin conocido como el modelo Lotka -Volterra, se ocupa de la interaccin entre dos especies, donde una de ellas (presa) tiene abundante comida, y la segunda especie (depredador) tiene suministro de alimentos exclusivamente a la poblacin de presas

# parametros del modelo
a = 0.7;  # Tasa de crecimiento de la presa
b = 0.5;  # Tasa de depredacin
c = 0.3;  # Tasa de mortalidad del depredador
e = 0.2   # Eficiencia de depredacin
dt = 0.001 # Diferencia de tiempo
tiempo_max = 100 # Tiempo maximo a alcanzar

# Tiempo inicial y poblacion
t = 0;
x = 1.0 #Presas
y = 0.5 #Depredadores

#listas vacas en las que almacenar el tiempo y las poblaciones
lista_t = [] #Lista de tiempo
lista_x = [] #Lista de Presas
lista_y = [] #Lista de Depredadores

# Inicialisa las listas
lista_t.append(t); lista_x.append(x); lista_y.append(y)

while t < tiempo_max:
    # Calcula los nuevos valores de t, x, y
    t = t + dt
    x = x + (a*x - b*x*y)*dt
    y = y + (-c*y + e*x*y)*dt

    # Agrega los valores en las listas
    lista_t.append(t)
    lista_x.append(x)
    lista_y.append(y)

# Muestra las graficas
p = plt.plot(lista_t, lista_x, 'r', lista_t, lista_y, 'g', linewidth = 2)
